With Vicodes you can add all sorts of essential elements to your any page easily.

== Description ==

Vicodes provides an essential set of easy to use shortcodes like buttons, alert boxes, tabs, toggles and accordion etc to make your life easier.

= List of Shortcodes =

* Alert Box
* Tabs
* Toggles
* Accordion
* Buttons
* Column/Grid
* Dividers
* Progress Bar
* Google Map
* Youtube Video
* Dailymotion Video
* Vimeo Video

== Installation ==

= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'Vicodes'
3. Activate Vicodes from your Plugins page.

= Install via FTP =
1. Unzip (unarchive) Vicodes zip file.
2. Upload 'vicodes` unzipped folder in ftp site-root/wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
